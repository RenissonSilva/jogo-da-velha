# Jogo da velha

Game criado em HTML, CSS e Node.js do famoso jogo da velha.

 ![](img/screen.png)
 
 
 ## Como jogar?
 Basta executar na pasta raiz do projeto o comando a seguir 
 
 ```sh
node server.js
```

E acessar no seu navegador o endereço

 ```sh
http://localhost:3000/
```
